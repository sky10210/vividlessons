# Vivid Lessons Module Publishing Checklist

Use this checklist before publishing any new Vivid Lessons module.

## Metadata

- [ ] MODULE_META is included near the top of the file
- [ ] Module ID is permanent and lowercase with hyphens
- [ ] Title is clear and teacher-friendly
- [ ] Category is correct
- [ ] Topic is correct
- [ ] Short description is one clear sentence
- [ ] Question count matches the actual number of scored questions
- [ ] Section count matches the actual number of module sections
- [ ] Primary question types are accurate
- [ ] Difficulty is Intro, Standard, or Advanced
- [ ] Shopify URL is added or marked as placeholder
- [ ] Preview URL is added
- [ ] Launch URL is added
- [ ] Standards field is left ready for future alignment

## Required Module Structure

- [ ] Student start form included
- [ ] First Name field included
- [ ] Last Name field included
- [ ] Class Period / Block field included
- [ ] Locked vertical sections included
- [ ] No auto-scroll between sections
- [ ] Every scored activity has a data-question tag
- [ ] Every scored activity has a data-type tag
- [ ] Reflection is required
- [ ] Certificate is included
- [ ] Score displays correctly
- [ ] Percent displays correctly
- [ ] Time started displays correctly
- [ ] Time finished displays correctly
- [ ] Total time displays correctly

## Question Rules

- [ ] Question count does not include reflection unless reflection is graded
- [ ] Multiple choice answers are similar in length
- [ ] Correct answers are not obvious by wording
- [ ] Questions use real scenarios when possible
- [ ] Feedback appears after each scored question
- [ ] Student cannot skip locked sections
- [ ] Final certificate only unlocks after required reflection

## Branding

- [ ] Uses Vivid Lessons branding
- [ ] Includes tagline: Learn. Apply. Succeed.
- [ ] Uses Poppins font
- [ ] Uses navy, blue, cyan, emerald palette
- [ ] No Mr. D, MrDHQ, or old branding appears
- [ ] Mobile layout looks clean
- [ ] Buttons and cards match current site style

## Completion Tracking

- [ ] Completion row follows standard format
- [ ] First name is recorded
- [ ] Last name is recorded
- [ ] Class period is recorded
- [ ] Module ID is recorded
- [ ] Module title is recorded
- [ ] Category is recorded
- [ ] Score is recorded
- [ ] Total questions is recorded
- [ ] Percent is recorded
- [ ] Question types are recorded
- [ ] Answers are recorded
- [ ] Time started is recorded
- [ ] Time finished is recorded
- [ ] Total time is recorded
- [ ] Reflection is recorded

## Product Card Readiness

- [ ] Homepage/product card title matches MODULE_META
- [ ] Product card description matches MODULE_META
- [ ] Product card shows question count
- [ ] Product card shows main question type
- [ ] Product card shows difficulty
- [ ] Preview button works
- [ ] Description button opens section list
- [ ] Buy on Shopify button works or uses placeholder
- [ ] Launch button points to dashboard or launch page

## Final Test

- [ ] Module opens on desktop
- [ ] Module opens on phone
- [ ] Start form works
- [ ] All sections unlock correctly
- [ ] All questions score correctly
- [ ] Reflection required before certificate
- [ ] Certificate generates correctly
- [ ] Completion tracking is ready or placeholder is clearly marked
